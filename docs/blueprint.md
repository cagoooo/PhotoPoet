# **App Name**: PhotoPoet

## Core Features:

- Poem Generation: Analyze the image and generate a poem that reflects its content, mood, and key elements, using a generative AI tool.
- Photo Submission: Provide a simple interface for users to upload or submit a photo URL.
- Poem Display: Display the generated poem alongside the submitted photo in a visually appealing layout.

## Style Guidelines:

- Primary color: Neutral white or light gray for clean presentation.
- Secondary color: Soft, desaturated blue or green to evoke calmness and creativity.
- Accent: A warm, earthy tone like terracotta or burnt orange to highlight key elements.
- Clean and readable sans-serif font for the main text.
- Clean, minimalist design with plenty of white space to focus on the image and poem.
- Subtle fade-in animations for the poem text.

## Original User Request:
An app that generates poems from photos
